# Local Images Directory

Place all your local images (e.g., PNG, JPG, SVG, WebP) in this folder.

To use them in your code, they are served relative to the root url `/`.
For example, if you place `my-image.png` in this directory:
- You can reference it as `/images/my-image.png`
- Or use the `ASSETS.images` object from `lib/assets.ts`.
