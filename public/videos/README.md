# Local Videos Directory

Place all your local videos (e.g., MP4, WebM) in this folder.

To use them in your code, they are served relative to the root url `/`.
For example, if you place `hero-bg.mp4` in this directory:
- You can reference it as `/videos/hero-bg.mp4`
- Or use the `ASSETS.videos` object from `lib/assets.ts`.
